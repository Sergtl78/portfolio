# Test task crypto-wallet on NextJS

Implemented on NextJS, MUI, Redux uses Chrome extension Metamask.

This project is used and implemented in this project:

- Created with TypeScript, NextJS, Redux, MUI, Chrome extension Metamask.
- The wallet address, wallet balance in ETH network and in BNB network are displayed.
- You can switch between accounts through the application.
- Realized the transfer between Sepolia and BNB Smart Chain Testnet accounts.
- Switching from the application between these two networks.
- Restricted access to other networks.

This project is scalable.
