# Landlord's Personal Account Project

This project uses and implements:

- Created with TypeScript, React, Next.
- CSS with Tailwind.
- Light and dark themes implemented with NextThemes.
- Forms using React-Hook-Form and Zod.
- Animation with Framer Motion.
