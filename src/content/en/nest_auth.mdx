# NestJS authentication example

Implemented on NestJS using JWT, as well as with support for Yandex and Google
authentication

This project used and implemented:

- Built with TypeScript, NestJS, Prisma, PostgreSQL, Docker.
- Supports Yandex and Google authentication.
- JWT strategy using asses and refresh token

This project can be scaled
