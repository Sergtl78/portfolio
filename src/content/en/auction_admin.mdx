# Project Admin panel of the auction

Admin panel for the auction of banner locations for the landlord's LC
b2b.delispace.ru

This project uses and implements:

- Created by fullstack using TypeScript, React, Next, Prisma, Postgres.
- Authentication using Next-auth.js
- CSS using Tailwind.
- The light and dark theme are implemented using NextThemes.
- Forms using React-Hook-Form and Zod.
