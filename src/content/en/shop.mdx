# Online store project

A small online store was developed in TypeScript, React, Next and TailwindCSS;
CMS Strapi was used for content management

This project used and implemented:

- Created with TypeScript, React, Next.
- CSS using Tailwind.
- Light and dark theme implemented using NextThemes.
- Authentication and shopping cart are implemented using Redux.
- GraphQL is used to interact with Strapi CMS.
- The project can be deployed using Docker.
- SEO can be edited via CMS.
- You can use the CMS to edit blocks on the main page, add product categories,
  administrate reviews, etc.

This project can be scaled.
