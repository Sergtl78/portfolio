# Пример аутентификации на NestJS

Реализовано на NestJS с помощью JWT, а также с поддержкой Яндекс и Гугл
аутентификацией.

В этом проекте использовано и реализованно:

- Создан с помощью TypeScript, NestJS, Prisma, PostgreSQL, Docker.
- Поддерживает Яндекс и Гугл аутентификацию.
- JWT стратегия с помощью asses и refresh token

Этот проект можно масштабировать.
