# Проект интернет-магазин

fioriopt.ru - закрытый сайт для оптовой продажи цветов. Приложение fullstack на NextJS, Typescript, Zustand, css - Tailwind, UI - shadcn.ui, БД - Postgres, ORM - Prisma.  Для управления написал CMS и CRM под нужды заказчика. CI CD реализовал через githab.

В этом проекте использовано и реализовавно:

- Создан с помощью TypeScript, React, Next.
- CSS с помощью Tailwind.
- UI Shadcn.ui
- Светлые и темные темы реализовавны с использованием NextThemes.
- Аутентификация с помощью Ayth.js : Yandex, Google, с email
- State manager Zustand.
- CMS и CRM под задачи заказчика
- Хранение, загрузка и использование фото товаров реализовано через YandexS3 
- CI CD реализовал через githab
