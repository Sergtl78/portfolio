# Проект Админ-панель аукциона

Админ-панель аукциона мест баннеров для ЛК арендодателя b2b.delispace.ru

В этом проекте использовано и реализованно:

- Создано fullstack с помощью TypeScript, React, Next, Prisma, Postgres.
- Аутентификация с помощью Next-auth.js
- CSS с помощью Tailwind.
- Светлая и темная тема реализованны с использованием NextThemes.
- Формы с использованием React-Hook-Form и Zod.
