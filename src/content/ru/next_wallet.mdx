# Тестовое задание крипто-кошелек на NextJS

Реализовано на NextJS, MUI, Redux использует Chrome extension Metamask.

В этом проекте использовано и реализовавно:

- Создан с помощью TypeScript, NextJS, Redux, MUI, Chrome extension Metamask.
- Отображается адрес кошелька, баланс кошелька в ETH сети и в BNB сети.
- черезерез приложение можно переключаться между аккаунтами.
- Реализовал перевод между аккаунтами в Sepolia и BNB Smart Chain Testnet.
- Переключение с приложения между этими двумя сетями.
- К другим сетям ограничил доступ.

Этот проект можно масштабировать.
