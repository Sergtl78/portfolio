# Проект портфолио

Здесь расказано о моих навыках и примеры моих работ с использованием TypeScript, React, Next и TailwindCss.

В этом проекте использовано и реализовавно:

- Портфолио создано с помощью TypeScript, React, Next.
- CSS с помощью Tailwind.
- Светлая и темная тема реализовавны с использованием NextThemes.
- На двух языках с использованием i18n, можно легко добавить поддержку еще одной языковой версии.
- Анимация с помощью Framer Motion.
- Описание проектов через .mdx файлы, чтобы удобнее редактировать.

Возможно далее это портфолио перерастет в блог.
