# Проект интернет-магазин

Небольшой интернет-магазин разработан на TypeScript, React, Next и TailwindCSS, для управления контентом использовано CMS Strapi

В этом проекте использовано и реализовавно:

- Создан с помощью TypeScript, React, Next.
- CSS с помощью Tailwind.
- Светлая и темная тема реализовавны с использованием NextThemes.
- Аутентификация и корзина реализовавны с помощью Redax.
- GraphQL используется для взаимодействия со Strapi CMS.
- Проект можно развенуть с помощью Dockker.
- SEO можно редактировать через CMS.
- Можно с помощью CMS редактировать блоки на главной странице, добавлять категории товаров, администрировать отзывы и т.д.

Этот проект можно масштабировать.
